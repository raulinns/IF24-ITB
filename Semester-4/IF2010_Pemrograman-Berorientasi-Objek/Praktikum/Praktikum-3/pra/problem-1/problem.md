Anda diminta untuk mengimplementasikan sebuah class template bernama Box<T> yang dapat menyimpan satu nilai bertipe generik.

Class Box<T> harus memiliki:

Constructor tanpa parameter
Constructor dengan parameter
Method untuk mengubah dan mengambil nilai
Method untuk menampilkan nilai dalam format tertentu
Selain itu, Anda juga diminta untuk mengimplementasikan sebuah function template bernama getMax yang mengembalikan nilai terbesar dari dua parameter bertipe sama.

Untuk mengetes program yang Anda buat, compile dan jalankan file main.cpp. Pastikan Anda hanya mengumpulkan jawaban lengkap dari Box.hpp saja. Selain itu, agar Anda dapat dengan mudah mengetes hasilnya, cocokkan outputnya sesuai dengan file output tersebut.
